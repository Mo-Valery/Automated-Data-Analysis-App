Steps to run the app:
1. Open CMD and make sure Python 3.8+ is installed. You can check by typing: python --version
2. Install Required Modules by pasting: pip install streamlit pandas matplotlib seaborn plotly openpyxl
3. cd into the directory with the app.py file eg. cd C:\Users\Desktop\LD7237_Project_Code (here you will simply copy the path of the folder where the file is)
4. Run the app in cmd by typing streamlit run app.py
5. Press enter and wait for the app to open on your browser
6. On your browser you can upload any excel file with sales data and with a date column. 
7. The analysis will now be done automatically and you will be able to see the visuals and insights
To clean the data just select the clean data automatically button
